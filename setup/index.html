    # Installation

    You can install the Arctic Black in many ways. We have listed some of them:

    # Automatic Installation

    1. Log in to your WordPress dashboard and go to Appearance > Themes.
    2. Type Arctic Black on theme search form.
    3. After the theme reveal on the page. Click Install and activate.

    # WP-CLI Installation

    Under your root directory of WordPress installation theme, you can type:

    wp theme install arctic-black

    # Plugin recommendation

    In order to get all features that Arctic theme can give. It is recommended to Install these plugins below:

    1. WP Term Images - This plugin allows users to assign images to any visible category, tag, or taxonomy term using the media library, providing a customized look for their taxonomies.
    2. WP Instagram Widget - This plugin is a no fuss WordPress widget to showcase your latest Instagram pics. It does not require you to provide your login details or sign in via oAuth.

    # Optional:

    1. Twitter Widget by Campaign Kit - Looking for the easiest way to display tweets on your WordPress site? The Campaign Kit Twitter widget allows you to display a stream of your recent tweets, with images hash and cash tags and a link to the original tweet. You can also display the Twitter account banner and a follow me link.
